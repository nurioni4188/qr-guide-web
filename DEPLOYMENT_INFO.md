# 방문 민원 QR 길잡이 - 배포 정보

## 배포 URL
**https://qrguide-9w5wlt9m.manus.space**

## 기능 특징

### ✅ 완성된 기능
- 로그인 없이 바로 접속 가능한 공개 정적 웹페이지
- 6개 민원 유형 안내 (산재보상, 피보험자격, 고용보험, 증명서, 보험료, 온라인서비스)
- 각 민원별 상세 정보 (필수서류, 체크리스트, 절차, 온라인 안내)
- 모바일 반응형 디자인 (QR 스캔 후 바로 사용 가능)
- 방문 민원인을 위한 단순하고 직관적인 UI

### ❌ 제거된 기능
- Manus OAuth 인증
- 사용자 로그인/회원가입
- 가이드 저장 기능
- 안내문 다운로드 기능
- 데이터베이스 저장 기능
- tRPC 서버 API

## 배포 특성
- 순수 정적 웹페이지 (React 정적 빌드)
- 서버/DB 불필요
- 무료 배포 (Manus 플랫폼)
- 모바일 최적화
- 빠른 로딩 속도

✅ 공개 URL 무로그인 접속 성공
✅ 6개 민원 유형 정상 표시
✅ 모바일 반응형 디자인 확인

## QR 코드

## 사용 방법
1. 방문 민원인이 QR 코드를 스캔
2. 배포 URL로 바로 이동
3. 민원 유형 선택
4. 필요한 정보 확인
5. 온라인 신청 링크로 이동 (선택사항)
