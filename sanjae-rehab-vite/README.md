# 산재보험 보상·재활 서비스 안내

React + TypeScript + Vite 기반의 IBM Carbon 스타일 공공기관 안내 웹서비스 예시입니다.

## 실행 방법

```bash
npm install
npm run dev
```

## 빌드

```bash
npm run build
npm run preview
```

## 구조

```text
├─ package.json
├─ index.html
├─ tsconfig.json
└─ src/
   ├─ App.tsx
   ├─ main.tsx
   ├─ types.ts
   ├─ data.ts
   └─ index.css
```
